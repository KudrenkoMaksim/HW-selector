Універсальний селектор

Універсальний селектор стосується кожного єлементу сторінки.
Зазвичай записується наступним чином:

* {
margin: 0;
padding: 0}

Селектор класу

Селектори класу використовуються для визначення стилів для кількох елементів одного типу, розміщених у різних частинах або на різних сторінках веб-сайту. Для створення заголовка класу headline необхідно додати атрибут class з відповідним значенням в відкриваючий тег. Далі необхідно задати стиль для зазначеного класу. Даний стиль оформлення можна застосувати і для інших елементів.
Приклад запису селектора класу:

.bold {
font-weight: 900;}

Селектор елемента

Селектор єлемента відповідає всім єлементам вказанного тега.
Приклади селекторів елементів:

h1 {
font-family: Lobster, cursive;}
p {
letter-spacing: 0.1em;}

Селектор атрибуту

Цей селектор вибирає всі єлементи, які мають вказаний атрибут або атрибут з певним значенням. Приклад запису:

Input [type= "text"] {
color: red;}

Селектор ідентифікатора

#first_id {
font-weight: bold;}

Селектор псевдоелементу

Псевдоелемент - це ключове слово, яке добавляється до селектора, дозволяє стилізувати певну частину вибраного єлемента.
Приклад:

Input::placeholder {
color:red}

Селектор псевдокласу

Псевдоклас - це ключове слово, добавлене до селектора, яке характеризує його особливий стан. Найбільш вживаний псевдокласу це :hover, який змінює поведінку об'єкта при наведенні на нього курсору.
Приклад запису:

div :hover{
background-color: #009688;}
